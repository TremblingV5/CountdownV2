# 倒计时Chrome插件

这是一个简单的Chrome浏览器倒计时插件，可以帮助您进行时间管理。

## 功能特性

- 设置小时、分钟和秒
- 开始、暂停和重置倒计时
- 倒计时结束时显示提示

## 安装方法

1. 打开Chrome浏览器
2. 输入 `chrome://extensions/` 进入扩展管理页面
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹

## 使用说明

1. 点击浏览器工具栏中的插件图标
2. 设置您需要的倒计时时间（小时、分钟、秒）
3. 点击"开始"按钮启动倒计时
4. 可以随时暂停或重置倒计时
5. 倒计时结束时会显示提示信息

## 技术栈

- HTML5
- CSS3
- JavaScript
- Chrome Extensions API
